# Detector Facial

Sistema de reconocimiento facial orientado a la detección de expresiones faciales en niños mediante visión por computadora y procesamiento de imágenes.

## Características
- Detección facial en tiempo real.
- Reconocimiento de expresiones faciales.
- Uso de cámara web para captura de video.
- Procesamiento automático de imágenes mediante Python.

## Tecnologías utilizadas
- Python 3.11
- OpenCV
- Librerías incluidas en `requirements.txt`

---

# Instalación

## 1. Instalar Python
Descargar e instalar Python 3.11 desde:
https://www.python.org/

---

## 2. Descargar el proyecto
Descomprimir el archivo `.zip` en cualquier carpeta de la computadora.

---

## 3. Abrir la terminal
Entrar a la carpeta del proyecto: `cd ruta\donde\descomprimio\detector_facial`

---

## 4. Crear entorno virtual
Comando: `py -3.11 -m venv venv 

---

## 5. Activar entorno virtual
Activarlo desde la terminal con: venv\Scripts\activate

---

## 6. Instalar dependencias
Instalar todas la librerias automaticamente: `pip install -requirements.txt`

---

## 7. Ejecutar el proyecto
En la terminal correr el proyecto: `python main.py`
