# detector_facial
***Sistema de reconocimiento facial de expresiones para niños***

Instalación
1. Instalar Python 3.11 desde python.org
2. Descomprimir el ZIP en una carpeta de su computadora
3. Abrir la terminal y entrar a la carpeta cd ruta\donde\descomprimio\detector_facial
4. Crear el entorno virtual: py -3.11 -m venv venv
5. Activarlo: venv\Scripts\activate
6. Instalar todas las librerias automaticamente: pip install -r requirements.txt
7. Correr el proyecto: python main.py
