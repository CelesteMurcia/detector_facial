# detector_facial
Sistema de reconocimiento facial de expresiones para niños

Requisito
 - Python 3.11

Instalación
***Paso 1. Instalar Python 3.11 desde python.org***
Paso 2. Descomprimir el ZIP en una carpeta de su computadora
Paso 3. Abrir la terminal y entrar a la carpeta cd ruta\donde\descomprimio\detector_facial
Paso 4. Crear el entorno virtual: py -3.11 -m venv venv
Paso 5. Activarlo: venv\Scripts\activate
Paso 6. Instalar todas las librerias automaticamente: pip install -r requirements.txt
Paso 7. Correr el proyecto: python main.py
